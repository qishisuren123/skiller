# DNA Methylation Beta-Value Analysis and DMR Detection

Create a command-line tool that processes DNA methylation beta-value arrays to identify differentially methylated regions (DMRs) between case and control samples.

DNA methylation beta-values represent the proportion of methylated cytosines at CpG sites, ranging from 0 (unmethylated) to 1 (fully methylated). Your tool should analyze these values across genomic positions to find regions with significant differential methylation patterns.
