# Key Libraries and Functions

## scipy.fft
- `fft(x)`: Compute FFT of signal x
- `dct(x, axis=0, norm='ortho')`: Discrete Cosine Transform

## numpy
- `np.hanning(N)`: Generate Hanning window of length N
- `np.log10(x)`: Base-10 logarithm
- `np.linspace(start, stop, num)`: Generate evenly spaced numbers
- `np.diff(x)`: Calculate discrete differences
- `np.sign(x)`: Element-wise sign of array

## argparse
- `ArgumentParser()`: Create argument parser
- `add_argument()`: Add command line arguments
- `parse_args()`: Parse command line arguments

## File I/O
- `np.load()`: Load .npz files
- `csv.writer()`: Write CSV files
- `json.dump()`: Write JSON files
